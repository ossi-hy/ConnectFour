# ConnectFour

## Asennus
Tarvitset [Poetryn](https://python-poetry.org/) asentaaksesi ohjelman.

Asenna välttämättömät riippuvuudet
```bash
$ poetry install
```
Aja ohjelma
```bash
$ poetry run invoke start
```
